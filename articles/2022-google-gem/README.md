# イベントに登壇する経緯

私の所属するチームでは Google Cloud の方にレビューやコンサルティングをお願いしています。
そのため、設計を共有している関係で我々が使っていた GKE + Agones が今回のイベントにピッタリだったので登壇の依頼をいただきました。
